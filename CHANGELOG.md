# 更新记录

本文件记录已同步到 GitHub 的项目版本变化。日期以 Asia/Shanghai 为准。

## 2026-07-19

### 新增

- 将研究样本从单一股票扩展为固定的 20 只 A 股，并为每只股票保存独立的日线行情文件。
- 为 14 个已注册因子生成独立的长表数据集，统一字段为
  `trade_date,trade_symbol,factor_value`。
- 增加 `CONTEXT.md`、`HANDOFF.md`、`PROJECT_ONBOARDING.md` 和两份 ADR，明确领域术语、数据契约、当前边界与接手方式。

### 变更

- 行情下载脚本改为批量下载固定研究股票池，使用固定窗口
  `2021-07-19` 至 `2026-07-16`，并保留 `amount` 字段。
- 行情字段统一为小写的 `trade_date`、`trade_symbol`、`open`、`high`、`low`、`close`、`volume`、`amount`。
- 因子计算改为遍历全部行情文件，并按“每个因子一份 CSV”的方式导出到 `data/factor/`。
- 使用当前因子代码和行情快照重新生成 14 份因子数据集，确保派生数据与实现一致。
- `CORR` 将无穷结果归一为缺失值，避免相关系数计算产生无穷值。
- 移除旧的单股票宽表因子结果，避免与当前 `data/factor/` 长表契约混用。
- 更新 README，使其反映当前多股票数据管道、数据布局和已知限制。

### 数据快照

- 20 份行情文件，合计 24,173 条 Market Data Observation。
- 14 份因子数据集，每份 24,173 条 Factor Observation。
- 日期范围为 `2021-07-19` 至 `2026-07-16`，交易日并集为 1,210 天。

### 验证

- 已通过 `python3 -m compileall -q factor.py factor_operators.py gat_data_akshare.py`。
- 已验证 20 份行情文件的列、日期排序、日期唯一性和证券代码一致性。
- 已验证 14 份因子文件的字段、排序、联合键唯一性、证券覆盖和行数。

### 已知限制

- 当前数据为探索性 `qfq` 快照，尚未完成严格的价格复权与 point-in-time 处理。
- 因子文件还没有补齐“日期并集 x 股票池”的完整观测网格。
- 已注册因子可以运行，但多数尚缺独立数值基准测试。
- `python3 -m pytest -q` 目前会因历史测试导入 `delta`、实现导出 `DELTA` 的不一致而在收集阶段失败。
